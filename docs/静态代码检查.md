
## 静态代码检查

本项目已配置了 **Ruff** 作为静态代码检查工具。

### 工具说明

**Ruff** - 快速且现代的 Python linter 和 formatter
- 集成了多种检查规则（pycodestyle, Pyflakes, isort 等）
- 支持代码格式化
- 速度极快，是传统工具的 10-100 倍

### 安装开发依赖

```bash
# 使用 uv（推荐）
uv sync --extra dev

# 或使用 pip
pip install -e ".[dev]"
```

### 使用方法

#### 在 Cursor 中使用

Cursor 会自动检测并集成这些工具，你可以在编辑器中看到：

1. **实时检查**：保存文件时自动显示错误和警告
2. **问题面板**：查看所有代码问题（`Ctrl+Shift+M` 或 `Cmd+Shift+M`）
3. **自动修复**：右键点击问题，选择 "Quick Fix" 或使用命令面板

#### 命令行使用

```bash
# 运行 Ruff 检查
ruff check .

# 运行 Ruff 格式化
ruff format .

# 检查格式（不修改文件）
ruff format --check .

# 自动修复可修复的问题
ruff check --fix .
```

### 配置说明

所有配置都在 `pyproject.toml` 中：

- `[tool.ruff]` - Ruff 的配置
- `[tool.ruff.lint]` - Ruff lint 规则配置
- `[tool.ruff.format]` - Ruff 格式化配置

可以根据项目需求调整这些配置。
